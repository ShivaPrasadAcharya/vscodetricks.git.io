F&R3: Making the text bold that lies between para tag and semicolon(:)

Find: (<p>)(.*?:)

Replace: <p><b>$2</b>

Explanation: (<p>): matches the starting <p> tag. (.*?:): matches any text until the first occurrence of a colon (:) and the colon itself, and captures this as the second group. <p><b>$2</b>: replaces the starting <p> tag with a new <p><b> tag, inserts the captured text from the second group with $2, and adds a closing </b> tag after the text.

Explanation:This will insert the pound symbol before the matched Devanagari script number. The $& in the replace string inserts the matched text (i.e., the number and dot).

F&R16:

Find:\w+)\s+(\w+)

Replace:$2, $1

Explanation :Here's an explanation of each term used in the regular expression (\w+)\s+(\w+):

(: This character starts a capture group. It tells the regular expression engine to "capture" the text that matches the pattern inside the parentheses.
\w+: This pattern matches one or more "word" characters. Word characters include letters (both uppercase and lowercase), numbers, and underscores.
): This character ends the capture group. It tells the regular expression engine to stop capturing text.
\s+: This pattern matches one or more whitespace characters. Whitespace characters include spaces, tabs, and line breaks.
(: This character starts a second capture group.
\w+: This pattern matches one or more word characters.
): This character ends the second capture group.
So putting it all together, the regular expression (\w+)\s+(\w+) matches two words separated by whitespace, and captures each word in its own capture group. The first word is captured by the first capture group, and the second word is captured by the second capture group.
For example, if you applied this regular expression to the string "John Smith", the first capture group would capture "John", and the second capture group would capture "Smith". You can then use these captured words in the replace string to rearrange the order of the text, as I showed in my previous answer.
